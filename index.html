Basics

Facts


dragon(falkor).
alignment(falkor, good).

dragon(smaug).
alignment(smaug, evil).

Rules


mother(tiamat, X) :- dragon(X), alignment(X, evil).

mother(daenerys, X) :- dragon(X).

Queries


?- mother(tiamat, falkor)
false.
            

?- mother(X, falkor).
X = daenerys.
            

?- mother(X, Y).
X = tiamat,
Y = smaug ;
X = daenerys,
Y = falkor ;
X = daenerys,
Y = smaug.

More examples

Parenting is simple


parent(abraham, isaak).
parent(isaak, jacob).
parent(jacob, judah).

ancestor(X, Y) :- parent(X, Y).
ancestor(X, Y) :- parent(X, Z), ancestor(Z, Y).
            

?- ancestor(abraham, judah).
true.

Parenting is hard


% Jan is Tom's wife
married(tom, jan).

% Jan is mother of Joy
mother(jan, joy).

% Joy has born Ray, from Tom's father Ben
mother(joy, ray).
father(ben, ray).

father_in_law(X, Y) :- married(X, Z), mother(Z, Y).
brother(X, Y) :- father(Z, X), father(Z, Y).
grandfather(X, Y) :- father_in_law(X, Z), mother(Z, Y).
grandfather(X, Y) :- grandfather(X, Z), brother(Z, Y).
            

?- grandfather(tom, tom).
true.
